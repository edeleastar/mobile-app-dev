Objectives

  • Download and explore the final versions of the donation applications

Donation-Service

Download the archive here:

  • donation-service-v2.zip

Unzip, and change into the folder containing the project.

Then run

play deps

and, once that completes:

play eclipsify

The import the project into Eclipse. It should import without errors. Then, run it:

play run

and explore the app at:

  • http://localhost:9000

Donation-Android

Download the android project here:

  • donation-android-v5

Unzip, and import into Eclipse.

It should build and execute successfully.

Semantic UI

Visit the semantic UI home page:

  • http://semantic-ui.com/

And download the archive. Explore the contents. The android-service project contains the contents of the 'packaged' folder (renamed to semantic).

There is also an 'examples' folder worth exploring.
